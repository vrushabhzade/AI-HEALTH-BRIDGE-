## 🌟 Overview

**HealthBridge** is an advanced AI-powered telemedicine and operational intelligence platform designed specifically for Nagpur's rural and district healthcare ecosystems. 

It serves a double purpose:
1. **Patient Telemedicine Hub:** Bridges the gap between rural patients and specialist doctors through AI diagnostics, appointment management, and secure video consulting.
2. **HealthBridge Operations (District Intelligence):** Enables real-time monitoring of Primary Health Centres (PHC) and Community Health Centres (CHC) across Nagpur district — incorporating automated demand forecasting, stock redistribution suggestions, attendance audits, diagnostics tracking, and AI-driven administrative escalations.

---

## ✨ Features

### 🏢 HealthBridge Operations (District & Facility Level)

#### 📊 PHC Staff Console
- **Real-Time Stock Monitoring:** Live inventory tracking with automated days-of-supply calculations.
- **Patient Footfall Tracker:** Records and visualizes daily department load against facility capacity.
- **Doctor Attendance Console:** Digitized shift logs tracking absences and tardiness.
- **Diagnostics Audit Panel:** Monitors status of laboratory equipment, reagents, and functional availability.
- **Bed Status Tracker:** Live occupancy rates for General and ICU beds.
- **AI Forecasting & Redistribution:** Integrated local mathematical models coupled with Gemini AI to generate narrative-style stock redistribution recommendations from surplus centers.

#### 🗺️ District Admin Dashboard
- **Nagpur Regional Interactive Map:** Direct SVG coordination mapping all active PHCs/CHCs with visual underperformance health states (Safe, Warning, Underperforming).
- **Composite Underperformance Index (0-100):** Weighted multi-factorial scoring algorithm blending stockouts, doctor absences, capacity overloads, and diagnostic downtime.
- **Automated Threat & Escalation Feed:** Live scan alerts flagging critical stockouts, capacity crises, and high absence rates.
- **One-Click Seeding:** Client-side database populator to instantly generate mock Nagpur facility telemetry for demo runs.

---

### 👤 For Patients & Doctors

#### 🤖 Premium Patient Dashboard & Vitals Tracker
- **Vitals Overview:** Sleek dark-mode grid cards for Glucose, BP, Heart Rate, and SpO₂ with animated status pills, range thresholds, and trend directions.
- **Interactive Vital Trends:** Chart.js powered timeline switching dynamically across multiple vital datasets.
- **Gemini AI Insight Banner:** Local context narrative generated by Gemini analyzing vital logs to provide wellness recommendations.
- **AI Symptom Checker:** Input symptoms in natural language for instant triage and emergency detection.
- **Weekly Step Activity Chart:** Interactive bar graph showing daily progress with color-coded target levels.

#### 👨‍⚕️ Telehealth & Consultations
- **Doctor Directory & Booking:** Browse 100+ verified professionals filtered by specialty.
- **Video Consultations:** Secure, HIPAA-compliant direct WebRTC sessions with screen sharing.
- **Lab Report Analyzer & Prescription Wallet:** Digital storage, medication reminders, and OCR-based report interpretations.

## 🛠️ Tech Stack

### Frontend
- **Framework:** React 19.2.0
- **Build Tool:** Vite 7.2.4
- **Routing:** React Router DOM 7.12.0
- **Animations:** Framer Motion 12.26.2
- **Charts:** Chart.js 4.5.1 + React-ChartJS-2
- **Icons:** Lucide React 0.562.0
- **Styling:** CSS3 with Custom Variables + Premium Glassmorphism Theme

### Backend & Database
- **Runtime:** Node.js + Express.js
- **Database:** Firebase Firestore
- **Authentication:** Firebase Auth
- **File Storage:** Firebase Storage

### AI Services
- **AI Engine:** Google Gemini API 0.24.1 (using Gemini 2.5 Flash model)
- **Functions:** Deterministic forecasting, underperformance scoring, narrative translation

## 🚀 Getting Started

### Prerequisites

Ensure you have these installed:
- **Node.js** (v18.0.0 or higher)
- **npm** (v9.0.0 or higher)
- **Firebase** account & **Google Gemini API** key

### Installation

1. **Clone the Repository**
   ```bash
   git clone https://github.com/vrushabhzade/AI-HEALTH-BRIDGE-.git
   cd AI-HEALTH-BRIDGE-
   ```

2. **Install Dependencies**
   ```bash
   npm install
   ```

3. **Configure Environment Variables**
   Create a `.env.local` file in the root directory:
   ```env
   # Firebase configuration
   VITE_FIREBASE_API_KEY=your_api_key
   VITE_FIREBASE_AUTH_DOMAIN=your_project.firebaseapp.com
   VITE_FIREBASE_PROJECT_ID=your_project_id
   VITE_FIREBASE_STORAGE_BUCKET=your_project.appspot.com
   VITE_FIREBASE_MESSAGING_SENDER_ID=your_sender_id
   VITE_FIREBASE_APP_ID=your_app_id

   # Gemini API Key
   VITE_GEMINI_API_KEY=your_gemini_api_key
   ```

4. **Run the Development Server**
   ```bash
   npm run dev
   ```
   Open `http://localhost:5173` in your browser.
